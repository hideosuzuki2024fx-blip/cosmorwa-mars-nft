# SpaceXアウトリーチ戦略（ミッション優先採用）

## Objectives
- Phase 0の採用を獲得し、参照標準としての価値を証明する。
- 土地販売やNFT誤読を排除し、ミッション価値を中心に据える。
- PRガードレールと3レイヤー構成に整合させる。

## Target Pathways
1. **経営陣への直接接触**（ミッション・エンジニアリング領域）。
2. **パートナー経由の紹介**（航空宇宙・地理情報・学術ネットワーク）。
3. **研究協力**（惑星科学・地図研究コミュニティ）。
4. **イベント参加**（運用・標準化に関する会合）。

## Action Plan (Specific Actions)
1. Phase 0停止点を強調した1ページ資料を作成。
2. 信頼できる航空宇宙アドバイザー経由で紹介を依頼。
3. Reference Gridの技術解説を公開。
4. 地理情報または惑星科学イベントで短時間発表。
5. 再現性検証に関する非公開ワークショップを開催。
6. Reference IDs付きのデモデータを提供。
7. 外部研究者との小規模パイロットを実施。
8. PR文言の事前承認を徹底。
9. SpaceX向けの中立なQ&A資料を用意。
10. 標準化団体や研究コンソーシアムからフィードバックを得る。
11. 着陸候補地検証に関する専門家ラウンドテーブルを実施。
12. 誤読防止チェックリストを提供。

## Alignment with PR Strategy
- 「NFTではなくミッション加速」が主メッセージ。
- 土地所有・不動産・主権に関する語彙を避ける。
- 収益化は付録のみで、初期アウトリーチに含めない。

## Expected Outcomes
- Phase 0採用可否の明確化。
- Reference Gridパラメータへの合意形成。
- 一貫したメッセージングによる評判リスクの低減。
