# SpaceXアウトリーチ戦略（採用向け）

## Objectives
- Phase 0（Reference-only）承認の獲得。
- 誤読防止を前提としたメッセージ統制。
- PR戦略との整合。

## Target Pathways
1. **経営陣への直接接触**（事業開発・ミッション責任者）。
2. **パートナー経由の紹介**（航空宇宙・学術パートナー）。
3. **研究協力**（科学・地図コミュニティ）。
4. **イベント参加**（宇宙政策・地理情報・科学会議）。

## Action Plan (10+ Specific Actions)
1. Layer 1とPhase 0停止点を示す1ページ資料を作成。
2. SpaceX関係者の紹介が可能なアドバイザー経由で連絡。
3. Reference Gridの技術解説ブログを公開。
4. 地理情報または惑星科学カンファレンスで短時間発表。
5. 研究者向けの非公開ワークショップを開催。
6. 大学や研究機関とReference IDの試験利用を実施。
7. Reference IDs付きのデモデータセットを提供。
8. PR文言の事前承認プロセスを整備。
9. SpaceXの公開データチームとのパイロット連携を提案。
10. 関連標準化団体へ技術ノートを投稿。
11. 検証手法に関する専門家ラウンドテーブルを実施。
12. 誤読防止チェックリストをSpaceXに提供。

## Alignment with PR Strategy
- 「SpaceXはNFTのためにロケットを打ち上げない」「土地所有ではない」を必ず明示。
- 収益化の話題は付録に限定。
- “reference grid”“reference IDs”を主要用語として統一。

## Expected Outcomes
- Phase 0採用可否の明確化。
- Reference Gridパラメータへのフィードバック。
- 誤読リスクの事前抑制。
