# PR戦略：誤読防止の徹底

## Core Framing
- SpaceXは**NFTのためにロケットを打ち上げない**。
- これは**土地所有・不動産・主権**ではない。
- 中核は**Mars Reference Grid**という座標標準。
- 収益化は付録に限定し、必須条件にしない。

## Do / Don’t List (15+)
1. **Do** 「reference IDs」「coordinate standard」を使用。
2. **Don’t** 「土地」「権利書」「不動産」を使わない。
3. **Do** Phase 0を停止点として強調。
4. **Don’t** 収益予測や市場機能を先に出さない。
5. **Do** 所有権や主権の否定を明示。
6. **Don’t** 投資リターンの示唆をしない。
7. **Do** 再現性と検証基準を説明。
8. **Don’t** 不動産市場との比較をしない。
9. **Do** 外部参加を研究貢献として説明。
10. **Don’t** 「購入」「転売」などの語彙を本文で使わない。
11. **Do** 収益化は付録に隔離。
12. **Don’t** SpaceXミッションとNFT販売を結びつけない。
13. **Do** NFTの有無に依存しない価値を強調。
14. **Don’t** 予測市場や賭博の話題を本文に入れない。
15. **Do** 誤解リスクへの注意喚起を含める。
16. **Don’t** 誇張表現や煽り表現を避ける。
17. **Do** データ共有の価値を強調。
18. **Don’t** SpaceXが外部マーケットを推奨する印象を与えない。
