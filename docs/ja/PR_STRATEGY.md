# PR戦略：ミッション優先・誤読防止

## Message Box
**NFTのためではなく、火星ミッション加速のため。** Reference Gridは位置参照の技術標準であり、SpaceXはNFTのためにロケットを打ち上げません。土地販売でもありません。

## Core Framing
- ミッション加速と検証済み候補地の提供が主価値。
- **土地所有・不動産・主権**ではない。
- Reference Gridは技術標準であり市場ではない。
- 収益化は任意で付録のみ。

## Do / Don’t List (Avoid misreads)
1. **Do** 「reference IDs」「coordinate standard」を使用。
2. **Don’t** 「土地」「権利書」「不動産」を使わない。
3. **Do** Phase 0停止点を強調。
4. **Don’t** 収益や市場規模を先に出さない。
5. **Do** 再現性と検証を強調。
6. **Don’t** 投資や投機の示唆をしない。
7. **Do** 外部参加を研究貢献として説明。
8. **Don’t** 不動産市場との比較をしない。
9. **Do** ミッションはトークンと無関係と明記。
10. **Don’t** 「購入」「転売」などの語彙を本文で使わない。
11. **Do** インセンティブは付録に限定。
12. **Don’t** 賭博・予測市場の話題を避ける。
13. **Do** 参照グリッドが曖昧性を減らす理由を説明。
14. **Don’t** SpaceXが外部市場を推奨する印象を与えない。
15. **Do** 誤読防止の注意書きを各プレス資産に入れる。
16. **Don’t** 誇張表現や利益強調を避ける。
