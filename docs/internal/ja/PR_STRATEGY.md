# 内部資料: ミッション・ファーストPR戦略（SpaceX採用）

## 中心メッセージ
NFTではなく、火星ミッションの加速が中心。検証、貢献整合、SpaceX主導の信頼を強調する。市場メカニズムは付録のみで任意。

## Do / Don’t リスト
1. **Do** ミッション加速と検証を最初に伝える。
2. **Do** 土地所有や主権の主張ではないと明記する。
3. **Do** SpaceXが基準と検証を完全に管理すると示す。
4. **Do** レイヤー1はトークン不要で成立すると強調する。
5. **Do** レジストリは標準化と説明責任の仕組みと位置づける。
6. **Do** 貢献は証拠に基づくと伝える。
7. **Do** 「参照のみ」「検証」「公開基準」の語彙を使う。
8. **Do** 付録は価値確認後にのみ言及する。
9. **Do** アクセスや打ち上げの約束がないことを明確にする。
10. **Do** 資金調達ではないと繰り返す。
11. **Don’t** 土地販売や領有権の話にしない。
12. **Don’t** 投機や利益を示唆しない。
13. **Don’t** 賭博や投資の言葉を使わない。
14. **Don’t** SpaceXが金融商品を出すと誤解させない。
15. **Don’t** 公式パートナーシップを勝手に主張しない。
16. **Don’t** 市場規模や収益予測を本文に入れない。
17. **Don’t** 検証より先に過度な話題化をしない。

## メッセージ構造
- **第一:** 検証可能な貢献によるミッション加速。
- **第二:** 誤情報の削減とエコシステム衛生。
- **第三:** 任意の仕組みは付録で扱う。

## 許容用語
- ミッション整合レジストリ
- 検証基準
- 参照のみフェーズ
- 公開承認
- 貢献証拠

## 典型的な誤解への回答
- 「NFTの話？」→「いいえ。ミッション経路はトークンを必要としません。」
- 「土地の所有？」→「いいえ。領有権や所有権はありません。」
- 「資金調達？」→「いいえ。検証のための枠組みです。」
